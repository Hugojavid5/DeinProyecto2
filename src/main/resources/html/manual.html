Gestión de Alumnos

  • Tabla de Alumnos

    Esta tabla muestra los datos de los alumnos registrados en el sistema.

    • DNI: El número de identificación del alumno

    • Nombre: El nombre del alumno

    • Primer Apellido: El primer apellido del alumno

    • Segundo Apellido: El segundo apellido del alumno

  • Filtros de Búsqueda

    Permite filtrar los alumnos por nombre utilizando un texto de búsqueda.

  • Botón Añadir

    Botón que permite agregar un nuevo alumno al sistema.

  • Botón Modificar

    Botón para modificar los datos de un alumno seleccionado.

Gestión de Libros

  • Tabla de Libros

    Esta tabla muestra los detalles de los libros disponibles en el sistema.

    • Código: El código único del libro

    • Título: El título del libro

    • Autor: El autor del libro

    • Editorial: La editorial que publicó el libro

    • Estado: El estado del libro (disponible, prestado, etc.)

    • Baja: Indica si el libro está dado de baja

    • Portada: La imagen de la portada del libro

  • Filtros de Búsqueda

    Permite buscar libros por título introduciendo un texto específico.

  • Botón Añadir

    Botón que permite añadir un nuevo libro al sistema.

  • Botón Modificar

    Botón para modificar los detalles de un libro seleccionado.

  • Botón Baja

    Botón para dar de baja un libro seleccionado.

Gestión de Préstamos

  • Tabla de Préstamos

    Esta tabla muestra el historial de préstamos de libros a los alumnos.

    • Id del Préstamo: El identificador único del préstamo

    • DNI del Alumno: El DNI del alumno que realiza el préstamo

    • Código del Libro: El código del libro que se presta

    • Fecha de Préstamo: La fecha en que se realiza el préstamo

    • Fecha de Devolución: La fecha en que se devuelve el libro

  • Botones de Filtro

    Permiten visualizar los préstamos filtrados por estado (devueltos, no devueltos, todos).

  • Filtros por DNI y Libro

    Filtros que permiten buscar préstamos por el DNI del alumno o el código del libro.

  • Botón Añadir

    Botón para registrar un nuevo préstamo en el sistema.

Gestión de Devoluciones

  • Tabla de Préstamos No Devueltos

    Esta tabla muestra los préstamos de libros que aún no han sido devueltos.

    • Id del Préstamo: El identificador del préstamo

    • DNI del Alumno: El DNI del alumno que tiene el préstamo

    • Código del Libro: El código del libro prestado

    • Fecha de Préstamo: La fecha en que se prestó el libro

  • Botón Devolver

    Botón para marcar un préstamo como devuelto y actualizar su estado.

Generación de Informes

  • Informe de Lista de Libros

    Botón que genera un informe con todos los libros registrados en el sistema.

  • Informe Gráfico

    Botón para generar informes visuales en forma de gráficos.

  • Informe de Cálculos

    Botón para generar informes detallados con cálculos sobre los datos de los alumnos y préstamos.
